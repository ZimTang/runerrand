# 基于 Koa.js 二次开发的跑腿小程序后台开发

技术栈为 `Koa + Sequelize + Mysql`

目前实现的功能为：

- [x] 用户相关
  - [x] 用户授权登录
  - [x] 用户信息修改
- [x] 分类（增删改查功能）
- [x] 地址（增删改查功能）
- [x] 订单
  - [x] 接单功能
  - [x] 增删改查功能
- [ ] 快递点（增删改查）
- [ ] 日志记录
- [ ] 参数验证器
- [ ] 代码规范化
